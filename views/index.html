URL Shortener Microservice

Example request

https://brainy-crate.glitch.me/new?url=https://github.com/mclintprojects/timestamp-microservice

Example response

{"original_url":"https://github.com/mclintprojects/timestamp-microservice","short_url":"https://brainy-crate.glitch.me/sXgs"}

Using https://brainy-crate.glitch.me/sXgs will now redirect to https://github.com/mclintprojects/timestamp-microservice.
